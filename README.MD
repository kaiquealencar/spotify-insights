# Spotify Insights

🎧 **Spotify Insights** é uma aplicação web feita em **Flask** que permite visualizar suas músicas e artistas mais ouvidos no Spotify de forma interativa. O projeto serve como vitrine para integração com APIs, mostrando **cards interativos** com previews das faixas e estatísticas de consumo musical.

---

## 🔹 Funcionalidades

- Login seguro com Spotify usando OAuth 2.0  
- Dashboard com seus **Top Artistas** e **Top Músicas**  
- Cards interativos com frente e verso, estilo Spotify  
- Reprodução das músicas mais ouvidas no último mês  
- Layout moderno e responsivo, inspirado no Spotify  

---

## 🔹 Tecnologias

- **Python 3.14.0** 
- **Flask**  
- **Spotify Web API**  
- **HTML5 / CSS3 / JavaScript**
